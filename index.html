Micro Radar Round TFT

Browser-based firmware installer for ESP32 using ESP Web Tools.

Flash Firmware

Connect your ESP32 board using a USB data cable and click the button below. Chrome or Microsoft Edge is required.

Before You Begin

  1. Connect the ESP32 using a USB data cable.
  2. Close Arduino IDE, PlatformIO or Serial Monitor.
  3. Click Flash Firmware.
  4. Select your serial port.
  5. Allow browser permissions.
  6. Wait until flashing completes.
  7. Press RESET if the board does not start automatically.

Important Notes

  • Use a USB data cable, not a charging-only cable.
  • Only Chrome and Microsoft Edge support Web Serial.
  • Disconnect other applications using the serial port.
  • If detection fails, manually enter bootloader mode.

Installation Steps

  1. Plug your ESP32 into your computer.
  2. Click the Flash Firmware button.
  3. Select your ESP32 serial port.
  4. Click Connect.
  5. Choose Install.
  6. Confirm that you want to erase the device if prompted.
  7. Wait until flashing reaches 100%.
  8. Disconnect and reconnect the board if required.

Browser Compatibility

ESP Web Tools uses the Web Serial API. For the best experience, use one of these browsers:

  • ✅ Google Chrome (Recommended)
  • ✅ Microsoft Edge
  • ❌ Firefox (Not Supported)
  • ❌ Safari (Not Supported)

USB Driver

If your ESP32 is not detected, install the correct USB driver for your development board.

  • CP210x Driver: Silicon Labs
  • CH340 Driver: Download
  • CH9102 Driver: WCH Official

Troubleshooting

Flash button cannot detect my ESP32
  • Use a USB data cable.
  • Reconnect the board.
  • Close Arduino IDE or any Serial Monitor.
  • Install the proper USB driver.
Failed to download manifest
  • Verify GitHub Pages is enabled.
  • Open manifest.json directly in your browser.
  • Check that firmware paths are correct.
  • Wait a few minutes after publishing GitHub Pages.
GitHub Pages shows 404
  • Confirm the repository name is correct.
  • Verify Pages is deployed from the correct branch.
  • Allow several minutes for the first deployment.
Manual Bootloader Mode
  1. Hold the BOOT button.
  2. Press and release RESET.
  3. Release BOOT.
  4. Click Flash Firmware again.
